Какая нейросеть использовалась

DeepSeek (основной инструмент) — 70% кода
ChatGPT (GPT-4) — 30% кода (отладка и оптимизация)

Какие части игры были созданы с помощью нейросети:

Полная архитектура игры — структура классов, игровой цикл
Система коллизий — алгоритмы обнаружения столкновений
Физика движения — гравитация, прыжки, скорость
Генерация уровня — размещение платформ, монет, врагов
Обработка ввода — управление с клавиатуры
Система камеры — плавное слежение за игроком
UI компоненты — кнопки, счетчики, оверлеи
Рендеринг графики — отрисовка персонажа, врагов, монет
Игровая логика — условия победы/поражения, система жизней
Оптимизация производительности — игровой цикл, интервалы обновления

Описание функционала:

HTML5 — структура страницы
CSS3 — стилизация, анимации, адаптивный дизайн
Vanilla JavaScript (ES6+) — вся игровая логика

Структура:

├── index.html          # Основной файл с Canvas и UI
├── style.css           # Стили для интерфейса
├── game.js             # Полная игровая логика
├── README.md           # Документация
└── screenshots/        # Папка со скриншотами
    ├── menu.png
    ├── gameplay.png
    └── win.png

Инструкция по запуску:

Движение:  ←  →  или  A  D
Прыжок:    ПРОБЕЛ  или  ↑  W
Рестарт:   R
Пауза:     кнопка на экране
Совет: Собирайте монеты на верхних платформах, чтобы избежать врагов! 😊
